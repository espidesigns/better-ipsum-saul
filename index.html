How To Use This Website

Need a quick HTML code with some placeholder text? You're in the right place. See the yellow titles down there? Press it to copy the code snippet to your clipboard. Then paste the code to your preferred code editor of choice. I'd highly recommend Sublime Text 2, but whatever floats your boat.

Toggling the switch on the right will show you how the codes will look on the browser, without the custom styles.
